How it works

Three steps. Each one falls back gracefully. Each output is real, not a hallucination.

1

Cluster by type

Documents are embedded with a hybrid structural+semantic signature, projected with UMAP, and clustered with HDBSCAN. No labels needed — the algorithm finds the doc types implicit in your folder.

2

Infer a schema

For each cluster, an LLM (Anthropic, OpenAI, Groq, Cerebras, Google, Ollama, OpenRouter) walks the documents and proposes fields with names, types, and example values.

3

Verify frequency

Every proposed field is re-checked across a spread of documents in the cluster, with grounding quotes. Fields with low frequency are flagged. The schema you get is one you can trust.

Why this exists

Every "AI document extraction" tool assumes you already know your schema. Real-world data isn't shaped that way.

Other tools start at extraction.

You point them at a doc with a schema you wrote by hand. If the doc is something you've never seen, you're stuck designing schemas by trial and error.

Lakshana starts at discovery.

You give it the folder. It tells you what's in there, then hands you back a clean schema per type. Then your extraction step can be simple.

Install

CPU-only, runs on a laptop. Pick whichever LLM provider you have a key for.

# minimum install
$ pip install lakshana

# with whichever LLM provider you use
$ pip install lakshana[openai]      # covers Groq, Cerebras, OpenRouter, Ollama
$ pip install lakshana[anthropic]   # Claude
$ pip install lakshana[google]      # Gemini
$ pip install lakshana[all]         # everything

# Python API
$ python
>>> import lakshana
>>> result = lakshana.discover(files=[...], model="groq/llama-3.3-70b-versatile")
>>> for cluster in result.clusters:
...     print(cluster["name"], result.schemas[str(cluster["id"])])

Why I built it

I've been building docstruct for a while — a full document extraction stack focused on Indian financial documents. The hardest part was never the extraction itself. It was figuring out, for any new client, what their documents even were.

So I built this. Lakshana is the "tell me what I have" piece, pulled out and standalone, so anyone building extraction can use it as a starting point.

If you're building anything in this space, I'd love to hear from you — open an issue with what you're trying to do.
