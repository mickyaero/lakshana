# Aircraft engineering sample set

30 synthetic aerospace documents across 3 types:

- **Maintenance Inspection Report (MIR)** — A/B/C/D-check findings, ATA-chapter referenced, fleet-realistic.
- **Flight Test Report (FTR)** — performance / handling / certification test runs per CAR-23 / CS-25.
- **Service Bulletin (SB)** — manufacturer-issued advisories, ATA-chapter referenced, with compliance + cost + tooling.

Generated synthetically by `generate.py` with a fixed seed for reproducibility. Used as a discover benchmark fixture, not training data.
